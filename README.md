# Code Description

This repository contains the code for the numerical examples for the paper "Linear-Quadratic Extended Mean Field Control Games" by Zhongyuan Cao, Mathieu Lauriere, and Jiefei Yang, 2026. (ADD ARXIV LINK WHEN AVAILABLE)

The code is relatively simple. There are two examples - one is a verification example ([LQMFGC-exp1.py](https://github.com/andrewjshi/2026-IEEE-CDC/blob/main/LQMFGC-exp1.py)) and one is a toy model ([LQMFGC-exp2.py](https://github.com/andrewjshi/2026-IEEE-CDC/blob/main/LQMFGC-exp2.py)).
